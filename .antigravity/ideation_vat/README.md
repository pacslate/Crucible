# Ideation Vat

This folder is the graveyard and launch pad for every idea we've considered.

**Every idea discussed in Ideation Mode is logged here — regardless of outcome.**
Ideas that make it to Stage 3 get their own project folder under `.antigravity/projects/`.

## Status Legend
- `EXPLORING` — Active conversation, verdict not yet reached
- `KILLED` — Didn't pass Stage 2 validation. Logged for future reference.
- `PIVOTED` — Original idea was modified. Pivot direction noted.
- `ACTIVE_PROJECT` — Made it to Stage 3. See `.antigravity/projects/` for the project folder.

## How to Browse

Each idea is a separate file named `YYYY-MM-DD_[idea-slug].md`. Browse this folder directly — each file is self-contained with the full Stage 1 and Stage 2 verdict inside.

> Ideas are created automatically during Ideation Mode at Stage 1 and updated through Stage 2.
